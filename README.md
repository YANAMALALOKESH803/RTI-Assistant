# RTI Assistant AI ⚖️

AI-powered CivicTech assistant built for RTI (Right to Information) guidance and draft generation.

## Features

AI-based RTI Question Answering
Retrieval-Augmented Generation (RAG)
RTI Draft Generator
Source PDF Access
Suggested Questions
Modern Dark UI
Streamlit Frontend

## Tech Stack

Frontend:
Streamlit

Backend:
Python
LangChain
HuggingFace
FAISS Vector Database
