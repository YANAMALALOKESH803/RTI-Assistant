Contributing to RTI Assistant AI
Thank you for your interest in contributing to RTI Assistant AI.
This project is part of a CivicTech initiative focused on simplifying access to Right to Information (RTI) services using Artificial Intelligence.
How to Contribute
We welcome contributions in the following areas:
Bug fixes
UI/UX improvements
RTI knowledge base enhancement
AI model improvements
Performance optimization
Documentation updates
Testing and deployment support
Development Setup
Clone the repository:
git clone <repository-url>
cd rti-assistant
Create a virtual environment:
python -m venv venv
Activate environment:
Windows Git Bash:
source venv/Scripts/activate
Install dependencies:
pip install -r requirements.txt
Run the application:
streamlit run app.py
Contribution Workflow
Fork the repository
Create a new branch
git checkout -b feature-name
Make changes
Test thoroughly
Commit changes
git commit -m "Describe your changes"
Push branch
git push origin feature-name
Open a Pull Request
Coding Guidelines
Please follow these guidelines:
Write clean and readable code
Use meaningful variable names
Keep functions modular
Add comments where necessary
Test features before committing
Maintain consistent formatting
Reporting Issues
If you find bugs or issues:
Clearly describe the issue
Mention steps to reproduce
Include screenshots or logs if possible
Feature Requests
Suggestions for improving RTI Assistant AI are welcome.
Possible contribution ideas:
Multilingual RTI support
Voice-enabled interaction
RTI tracking assistance
Government portal integration
Improved document retrieval
Code of Conduct
Please maintain respectful and collaborative communication while contributing.
Acknowledgements
